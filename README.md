# 🌑 Eclipse - Site Interativo

Este projeto é um **site interativo inspirado no livro Eclipse**, da saga *Crepúsculo*, escrito por Stephenie Meyer.  
O site apresenta personagens, times (Team Edward ou Team Jacob), história do livro e uma batalha interativa entre vampiros e lobos.

O objetivo do projeto é demonstrar o uso de **HTML, CSS e JavaScript** para criar uma experiência visual e interativa para o usuário.

---

# 📌 Funcionalidades

### 🧛 Tema Vampiro / 🐺 Tema Lobo
O usuário pode alternar entre dois temas do site:
- Tema Vampiro
- Tema Lobo

Ao trocar o tema:
- O **vídeo de fundo muda**
- O **estilo do site muda**
- O **ícone do botão muda**

---

### 📖 História do Livro
A seção **Sobre o Livro** apresenta:
- Um botão para abrir o livro
- Um efeito visual simulando **duas páginas abertas**
- Resumo e conflito da história

---

### 👥 Personagens
O usuário pode clicar nos personagens principais:

- Bella Swan
- Edward Cullen
- Jacob Black

Ao clicar:
- Aparece **imagem do personagem**
- Aparece **descrição com efeito de texto digitando**

---

### ⚔️ Escolha de Time
O visitante pode escolher entre:

- **Team Edward**
- **Team Jacob**

Ao escolher:
- Uma imagem aparece
- Um texto explicativo é exibido
- O tema do site muda de acordo com o time escolhido

---

### 🔥 Batalha Final
Existe uma batalha interativa entre:

- Vampiros
- Lobos

Cada lado possui:
- Barra de poder
- Botão de ataque

Ao atacar:
- A barra aumenta
- O site treme (efeito de batalha)
- Aparece dano e partículas
- Quando a barra chega a 100%, um vencedor é exibido

---

### 🩸 Efeito de Cursor
O site possui um efeito visual no cursor:

- Ao mover o mouse aparecem **gotas de sangue animadas**

---

### 🎬 Vídeo de Fundo
A página inicial possui um **vídeo de fundo automático**, que muda dependendo do tema escolhido.
